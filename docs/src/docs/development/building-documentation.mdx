## Introduction

The documentation is built using [NodeJS](https://nodejs.org/en/). The documentation source code is contained in the [docs/](https://github.com/LilDataMonster/loincloth/tree/master/docs) folder.

## Compile and Develop Page


### Install Dependencies

Install dependencies using yarn by running `yarn` in the location of the `package.json` file.

### Run Develop mode

Run the documentation webpage by running `yarn start` in the location of the `package.json` file. This will run a server with hot-reloading which will apply changes to the webpage when file updates are saved.
